X-ray Galaxy Cluster Archive

À l'Université de Montréal

Welcome to the X-ray Galaxy Cluster Archive à l'Université de Montréal! The archive was built out of several graduate students' desire to have a coherent and comprehensive data archive dedicated the X-ray Galaxy Clusters. While the primary purpose of the archive was initially to categorize clusters into cool-core vs. non cool-core, we hope it will enable further investigation into cluster properties and allow for statistical studies of clusters.


On the Cluster Table page, you can find a searchable collection of clusters along with pertinent information on each including parameters such as the cooling radius and coefficient of surface brightness. If you visit the Cluster Plots page, you will be able to create plots using the calculated cluster parameters and download the datatable as a csv file.
