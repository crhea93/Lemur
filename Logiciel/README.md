Readme file for Code and CSV files

Clusters.csv has several categorizes. While most are self-explanatory, several are not...
Stable: All stability is determined from X-ray temperature profiles
	-none -- cannot determine if stable or not
	-stable -- No cool core
	-cool -- clearly a cool core 
	-merger -- other literature source claims merger

Source_X: X-ray data source 
	-M -- data from Michigan State Archive (https://web.pa.msu.edu/astro/MC2/accept/)
	-P -- Data found in other paper

Source_L: Luminosity Source (e.g. Author;Telescope)
	-Author Name
	-Telescope

Mass_est: Type of Mass Estimate
	-W -- Weak Lensing
	-X -- X-ray Estimate
